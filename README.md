# Pitbull in Prague V2 — fixed build

Copy the contents of this folder into the root of your GitHub repository. The root should contain `index.html`, `manifest.webmanifest`, `service-worker.js`, `README.md`, and the `assets` folder.

After pushing, wait for GitHub Pages to deploy, then open the site in Safari. Remove the old Home Screen icon and add the site again so iOS does not retain the broken cached build.

The concert countdown uses a provisional 20:30 target because the actual Pitbull stage time is not yet confirmed. Doors remain shown as 15:00 and arrival remains TBC.
