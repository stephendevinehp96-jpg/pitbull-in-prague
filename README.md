# Pitbull in Prague

## Upload
Upload all files and folders in this ZIP to the root of the `pitbull-in-prague` GitHub repository.

## Publish with GitHub Pages
Repository **Settings → Pages → Deploy from a branch → main → /(root) → Save**.

Expected URL:
https://stephendevinehp96-jpg.github.io/pitbull-in-prague/

## Install on iPhone
Open the live site in Safari, tap **Share**, then **Add to Home Screen**.
